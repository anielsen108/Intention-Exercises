# Intimacy Gradients — Specialized Exercises

Focused exploration of relational closeness, vulnerability levels, and appropriate disclosure. Practice navigating from formal distance to deep intimacy.

---

## Formal Distance (1-15)

### 1. "Good morning."

**Professional courtesy**
- ToBI: L* L+H* L-L%
- Stress: Good **MORNing**
- Pause: Good morning.
- Neutral, appropriate distance.

**Warm colleague**
- ToBI: L+H* L+H* L-H%
- Stress: Good **MORNING**!

- Friendly but boundaried.

**Intimate greeting**
- ToBI: L+H* L+H* L-H%

## Practice Notes

- **Intimacy** in voice appears as softness, breathiness, slower pace, personal pronouns
- Distance uses clearer tone, faster pace, formal language
- Mismatched intimacy feels uncomfortable — practice calibrating
- Breath matters: shallow/chest for distance, deep/belly for intimacy
- Volume: softer generally signals closeness
- Some cultures value formal distance, others warm closeness by default
- Professional settings require managing intimacy carefully
- Vulnerability must match relational safety level
- Practice appropriate boundaries while staying human

## Integration

Use these exercises to:
1. Build genuine connection appropriately
2. Maintain professional boundaries
3. Deepen important relationships
4. Navigate cross-cultural interactions
5. Recognize and respond to others' intimacy signals
6. Protect yourself from premature vulnerability
7. Offer safe space for others' disclosure
